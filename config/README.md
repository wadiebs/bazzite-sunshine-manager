# Config folder

Environment-first configuration. You can override these by setting env vars
or by creating a local `.env` (if you wire in dotenv) or editing and exporting before running.

## Common toggles

- IMPORT_STEAM=1
- IMPORT_HEROIC=1
- INCLUDE_SYSTEM_APPS=1
- ADD_REBOOT_LAUNCHER=1
- SGDB_ENABLE=1
- SGDB_API_KEY=YOUR_KEY
- SGDB_TIMEOUT=12
- USE_DEFAULT_BLACKLIST=1
- BLACKLIST_IDS=""
- BLACKLIST_NAME_REGEX=""
- INCLUDE_SOURCES="epic,gog"
- SYSTEM_APPS_JSON="/usr/share/sunshine/apps.json"
- FORCE_COLOR=0
